Link đã deploy:

Cách chạy demo
	1. Copy nội dung trong thư mục 'source' vào trong 'htdocs' của XAMPP
	2. Sửa file dbconnection.php từ dòng 11 -> 14 cho phù hợp với cấu hình máy
	3. Import 'ql_ungdung.sql' vào phpMyAdmin
	4. Khởi động XAMPP và truy cập http://localhost
